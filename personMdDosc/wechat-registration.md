# 微信公众号注册指南

## 1. 注册订阅号

1. 打开 https://mp.weixin.qq.com
2. 点击"立即注册"
3. 选择"订阅号"
4. 填写信息：
   - **邮箱**：未注册过微信公众平台的邮箱
   - **密码**：登录密码
   - **验证码**：邮箱验证码

## 2. 账号类型选择

| 类型 | 说明 | 群发频率 | 适用场景 |
|------|------|----------|----------|
| **个人订阅号** | 用身份证注册 | 每周 1 次 | 个人使用 |
| **企业/个体户订阅号** | 用营业执照注册 | **每天 1 次** | 面向公众 |

> **建议**：如果要面向公众推送，注册**企业/个体户订阅号**，这样才能每天群发。
> 个人订阅号每周只能群发 1 次，不够用。

## 3. 完成认证

### 个人订阅号
- 上传身份证正反面照片
- 填写姓名、身份证号
- 微信实名认证（扫脸）

### 企业/个体户订阅号
- 填写企业名称、统一社会信用代码
- 上传营业执照照片
- 填写法定代表人信息
- 对公账户打款验证（企业）或微信支付验证（个体户）

## 4. 获取 API 凭证

认证通过后：

1. 登录公众号后台：https://mp.weixin.qq.com
2. 左侧菜单：**开发 → 基本配置**
3. 获取：
   - **AppID**（应用 ID）
   - **AppSecret**（应用密钥，点击"生成"后保存）

> ⚠️ AppSecret 只显示一次，务必立即保存！

## 5. 配置服务器域名

1. 左侧菜单：**开发 → 公众号设置 → 功能设置**
2. 配置"JS 接口安全域名"和"网页授权域名"
3. 对于我们的项目，需要配置 **GitHub Pages 域名**：
   - `jsapi_domain`: `yourname.github.io`
   - `web_domain`: `yourname.github.io`

## 6. 上传封面图

1. 准备一张封面图片（建议尺寸 900×500px，格式 JPG/PNG）
2. 左侧菜单：**素材管理 → 素材管理**
3. 上传封面图
4. 记录封面图的 `media_id`（后续推送需要）

## 7. 配置 GitHub Secrets

在 GitHub 仓库 Settings → Secrets and variables → Actions 中添加：

| Secret | 说明 | 示例 |
|--------|------|------|
| `WECHAT_APP_ID` | 公众号 AppID | `wx1234567890abcdef` |
| `WECHAT_APP_SECRET` | 公众号 AppSecret | `abc123def456...` |
| `WECHAT_COVER_IMAGE_PATH` | 封面图片路径 | `cover.jpg` |
| `OPENAI_API_KEY` | OpenAI API Key | `sk-xxx...` |

## 常见问题

### Q: 个人订阅号可以每天群发吗？
**A:** 不可以。个人订阅号每周只能群发 1 次。如需每天群发，必须注册企业/个体户订阅号。

### Q: 群发次数用完了怎么办？
**A:** 每天群发图文消息后，当天不能再群发。可以通过"草稿箱"先保存，到第二天再群发。

### Q: 推送内容会被审核吗？
**A:** 群发图文消息会经过微信内容审核。我们的推送内容仅含新闻标题和摘要，一般不会被拦截。

### Q: 粉丝怎么关注我的公众号？
**A:** 生成公众号二维码，分享到微信群/朋友圈，用户扫码即可关注。

### Q: 如何查看推送结果？
**A:** 左侧菜单：**统计 → 消息/图文统计** 可查看阅读量、转发量等数据。
